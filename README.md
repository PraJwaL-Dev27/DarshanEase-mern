# 🛕 DarshanEase – MERN Stack Temple Booking System

DarshanEase is a full-stack MERN application for managing temple darshan bookings, donations, feedback, and admin operations.

---

## 🚀 Features

- User Registration & Login
- Role-based Authentication (User / Organizer / Admin)
- Temple Creation & Management
- Slot Booking System
- Donation System
- Feedback & Rating System
- Admin Dashboard
- Secure JWT Authentication

---

## 🛠 Tech Stack

### Frontend
- React
- MUI (Material UI)
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
